**Q53. Answer: (b) O(n)**

Building a heap bottom-up using heapify (sift-down):

- Start from node at index floor((n−1)/2) down to index 0
- Cost of heapifying a node at height h = O(h)
- Total cost = sum over all nodes of O(height)
- Mathematical result: sum = O(n) × sum(h/2^h) = O(n) × O(1) = O(n)

Bottom-up heap construction is always O(n), not O(n log n).

**Q63. Answer: (a)**

Algorithm: if requesting process Pr has timestamp T(Pr) < T(Ph) (Pr is OLDER), kill Pr. Else Pr waits.

Is it deadlock free? YES.
- Circular wait cannot form because a younger process always waits or gets killed.
- The oldest process in any cycle always gets the resource. No cycle can be maintained.

Is it starvation free? NO.
- A young process can be killed and restarted repeatedly whenever it tries to get a resource held by an older process.
- It may never get the resource (starvation).

Answer: (a) Deadlock free but NOT starvation free.
